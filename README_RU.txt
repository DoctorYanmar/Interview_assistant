###Interview Assistant


Interview Assistant - это программа, которая помогает во время онлайн-интервью, предоставляя подсказки на основе вопросов интервьюера с использованием ChatGPT.


##Установка

#Windows

Скачайте и распакуйте архив с программой.
Запустите setup.bat, дважды кликнув на него.
Следуйте инструкциям на экране. Скрипт установит Python 3.10 (если его нет), создаст виртуальное окружение и установит необходимые зависимости.
После завершения установки откроется браузер с сайтом VB-Audio Virtual Cable. Скачайте и установите эту программу.

#macOS / Linux

Скачайте и распакуйте архив с программой.
Откройте терминал в папке с программой.
Выполните команду(macOS): `chmod +x setup_macos.sh && ./setup_macos.sh`
Выполните команду(Linux): `chmod +x setup_linux.sh && ./setup_linux.sh`
Следуйте инструкциям на экране. Скрипт установит Python 3.10 (если его нет), создаст виртуальное окружение и установит необходимые зависимости.
После завершения установки откроется браузер с сайтом VB-Audio Virtual Cable. Скачайте и установите эту программу.


##Настройка

#Настройка API ключа

Откройте файл .env в текстовом редакторе.
Замените your_api_key_here на ваш API ключ OpenAI.


Настройка виртуальных аудио выходов


#Windows

Шаг 1: Установка виртуального аудиоустройства

Скачайте и установите VB-Audio Virtual Cable с официального сайта: https://vb-audio.com/Cable/
После установки перезагрузите компьютер, чтобы убедиться, что все драйверы корректно установлены.
После перезагрузки у вас должно появиться новое аудиоустройство под названием "CABLE Input" и "CABLE Output". Это два конца одного виртуального кабеля.

Проверка установки:

Откройте настройки звука Windows (правый клик на значок динамика -> Звуковые настройки)
В разделе "Вывод" вы должны увидеть "CABLE Input"
В разделе "Ввод" вы должны увидеть "CABLE Output"

Шаг 2: Настройка Zoom

Откройте Zoom и войдите в свою учетную запись.
Нажмите на иконку шестеренки в правом верхнем углу, чтобы открыть настройки.
В левом меню выберите "Звук".
В разделе "Динамик" выберите "CABLE Input (VB-Audio Virtual Cable)".
В разделе "Микрофон" оставьте ваш обычный микрофон.

Шаг 3: Настройка Windows для прослушивания

Откройте настройки звука Windows (правый клик на значок динамика -> Звуковые настройки).
Прокрутите вниз до раздела "Дополнительные настройки звука" и нажмите на "Панель управления звуком".
Перейдите на вкладку "Запись".
Найдите в списке "CABLE Output" и дважды щелкните на нем.
Перейдите на вкладку "Прослушать".
Поставьте галочку напротив "Прослушивать это устройство".
В выпадающем списке "Воспроизводить через это устройство" выберите ваши наушники или динамики, через которые вы хотите слышать звук.
Нажмите "Применить", затем "ОК".

Проверка настройки:

Вернитесь в настройки Zoom.
В разделе "Динамик" нажмите "Проверить динамик".
Вы должны услышать тестовый звук через ваши наушники или динамики.


#macOS

Шаг 1: Установка виртуального аудиоустройства

Скачайте и установите VB-Audio Virtual Cable для macOS с официального сайта: https://vb-audio.com/Cable/
После установки перезагрузите компьютер.

Шаг 2: Настройка Zoom

Откройте Zoom и перейдите в настройки (zoom.us -> Настройки).
Выберите вкладку "Звук".
В разделе "Динамик" выберите "CABLE Input (VB-Audio Virtual Cable)".
В разделе "Микрофон" оставьте ваш обычный микрофон.

Шаг 3: Настройка macOS для прослушивания

Откройте "Системные настройки" -> "Звук".
Перейдите на вкладку "Выход" и выберите ваше обычное устройство вывода звука (наушники или динамики).
Перейдите на вкладку "Вход".
Убедитесь, что выбран ваш обычный микрофон.

Шаг 4: Настройка прослушивания виртуального устройства

Откройте приложение "Аудио MIDI Setup" (находится в папке "Программы" -> "Утилиты").
В меню "Окно" выберите "Показать окно аудиоустройств".
Найдите в списке устройств "CABLE Output" и выберите его.
Нажмите на кнопку с изображением шестеренки в нижнем левом углу окна.
Выберите "Использовать это устройство для вывода звука".
В появившемся диалоговом окне выберите ваше обычное устройство вывода звука.

Проверка настройки:

Вернитесь в настройки Zoom.
Нажмите "Проверить динамик и микрофон".
Вы должны услышать тестовый звук через ваши наушники или динамики.

Если вы не слышите звук или программа не захватывает аудио, проверьте настройки звука в системе и в Zoom. Убедитесь, что все устройства выбраны правильно и уровни громкости установлены на достаточном уровне.


##Использование программы

Запустите программу:

Windows: Дважды кликните на interview_helper.bat
macOS Запустите interview_helper_macos.sh (./interview_helper_macos.sh)
Linux: Запустите interview_helper_linux.sh (./interview_helper_linux.sh)



В открывшемся окне программы:

Выберите нужное аудиоустройство (обычно это будет "CABLE Output").
Выберите модель ChatGPT (например, "gpt-4o-mini").
Напишите промпт в окне промпта, распишите максимально точно требования для ассистента(в настройках по умолчанию есть промпт, можно использовать как эскиз)
Выберите язык (русский или английский).
Нажмите Save Settings для создания файла с настройками для дальнейшей корректной работы программы.


Когда интервьюер начнет задавать вопрос, нажмите кнопку "Start Recording".
Когда интервьюер закончит вопрос, нажмите кнопку "Stop Recording" (та же кнопка, что и "Start Recording").
Программа обработает аудио, преобразует его в текст и отправит запрос к ChatGPT.
В окне "ChatGPT Suggestions" появится подсказка для ответа на вопрос.

##Важные замечания

Убедитесь, что вы можете запустить "виртуальную сеть", если страница https://openai.com/ не открывается в вашей стране
Убедитесь, что вы правильно настроили виртуальные аудио выходы перед использованием программы.
Проверьте, что в файле .env указан правильный API ключ OpenAI.
Если вы не слышите звук или программа не захватывает аудио, проверьте настройки звука в системе и в Zoom.
Помните, что использование такой программы может быть неэтичным в некоторых ситуациях. Убедитесь, что вы не нарушаете правила проведения интервью.

##Устранение неполадок

Если программа не запускается, убедитесь, что вы выполнили все шаги установки.
Если не работает захват звука, проверьте настройки виртуальных аудио выходов.
При проблемах с распознаванием речи попробуйте говорить четче и медленнее.
Если ChatGPT не дает релевантных подсказок, попробуйте изменить модель или промпт в настройках программы.

###Получение API ключа OpenAI

Для работы программы необходим API ключ OpenAI. Чтобы получить его, выполните следующие шаги:

Перейдите на сайт OpenAI: https://openai.com/
Нажмите "Sign up" (если у вас еще нет аккаунта) или "Log in" (если аккаунт уже есть).
После входа в систему перейдите в раздел API: https://platform.openai.com/account/api-keys
Нажмите кнопку "Create new secret key".
Скопируйте сгенерированный ключ (важно сохранить его, так как он отображается только один раз).
Вставьте этот ключ в файл .env в директории программы, заменив your_api_key_here на ваш настоящий ключ.

Важно: Для использования API ключа необходимо привязать способ оплаты к вашему аккаунту OpenAI. После окончания бесплатного пробного периода (если таковой предоставляется) использование API будет тарифицироваться в соответствии с текущими расценками OpenAI.
Примечание: Храните ваш API ключ в безопасности и не делитесь им с другими. Использование API может привести к финансовым расходам, поэтому следите за использованием и установите лимиты расходов в настройках вашего аккаунта OpenAI.
